<h2> :gear: Executando a Aplicação </h2>

```bash

# Clone o repositório:
git clone https://github.com/EquipeGfour/Estacao-Metereologica-Front-end.git

# Navegue para o caminho:
cd front-end

# Instale as dependências:
npm install

# Inicie o sistema:
npm start
```
<br>
